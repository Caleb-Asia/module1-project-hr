# HR Dashboard Management System

##  Project Overview

The HR Dashboard Management System is a modern web-based application designed to streamline employee management, attendance tracking, payroll monitoring, and leave management within an organization. The system provides HR personnel and managers with a centralized platform to monitor workforce activities through an intuitive and responsive dashboard interface.

This project was developed as part of a web development module to demonstrate the implementation of front-end technologies, user interface design principles, and dynamic data management using JavaScript.

---

##  Features

### Dashboard Overview

* Interactive dashboard displaying key HR metrics.
* Quick access to employee statistics and company records.
* Modern dark-themed user interface with responsive design.

### Employee Management

* View employee records in a structured table.
* Add new employee information.
* Manage employee details efficiently.

### Attendance Tracking

* Record employee attendance.
* Track attendance status (Present, Late, Absent).
* Store attendance records dynamically.

### Payroll Monitoring

* View payroll-related information.
* Monitor employee payment records.
* Organize payroll data for HR administration.

### Leave Management

* Manage employee leave requests.
* Track approved and pending leave records.
* Maintain leave history for employees.

### User Interface

* Responsive layout suitable for desktop and mobile devices.
* Sidebar navigation for easy access to modules.
* Clean and modern design focused on usability.
